TITEL:
Beskatning af pension

KILDER:

* https://skat.dk/borger/pension-og-efterloen/skat-af-pensionsafkast

BESKRIVELSE:
Pension beskattes forskelligt afhængigt af pensionsordningstype og tidspunkt for beskatning (indbetaling eller udbetaling).

REGLER:

* Ratepension og livrente beskattes ved udbetaling
* Aldersopsparing beskattes ved indbetaling
* PAL-skat på afkast

BETINGELSER:

* Afhænger af pensionsordningstype
* Afhænger af kundens samlede indkomst

UDBETALING / FUNKTION:

* Skat trækkes automatisk ved udbetaling
* Indgår i personlig indkomst

SKAT:

* Rate/livrente → indkomstskat
* Aldersopsparing → skattefri udbetaling
* PAL-skat på afkast

FORDELE:

* Fradrag kan give skattemæssig fordel nu
* Mulighed for skatteoptimering

ULEMPER:

* Kan føre til høj beskatning ved udbetaling
* Påvirker folkepension og tillæg

RÅDGIVNING:

* Vurder skat nu vs senere
* Se pension i sammenhæng med øvrig indkomst
* Tilpas strategi efter kundens skatteprocent

FALDGRUBER:

* For høj samlet indkomst ved pension → høj skat
* Manglende planlægning → tab af tillæg
